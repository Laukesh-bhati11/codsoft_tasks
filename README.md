# CodSoft Web Development Internship Tasks

Welcome to my repository for the CodSoft Web Development Internship! This repository contains the three completed projects required for successful internship completion.

## 📁 Repository Structure

The repository is organized into distinct project folders:
*   **Level-1/Task-1-Portfolio/**: A personal portfolio website showcasing my profile, skills, and projects.
*   **Level-1/Task-2-LandingPage/**: A creative, well-aligned standalone landing page.
*   **Level-1/Task-3-Calculator/**: A fully interactive, responsive grid-based calculator.

---

## 🚀 Projects Overview & Features

### 1. Personal Portfolio Website
*   **Tech Stack:** HTML5, CSS3, JavaScript
*   **Features:**
    *   Responsive navigation header.
    *   About section with profile details.
    *   Skills and expertise grid.
    *   Downloadable PDF resume link.
    *   Interactive contact form.

### 2. Creative Landing Page
*   **Tech Stack:** HTML5, CSS3 (Tailwind CSS if applicable)
*   **Features:**
    *   Visually striking hero section.
    *   Clean layout alignments with precise padding.
    *   Fully mobile-responsive components.

### 3. Interactive Calculator
*   **Tech Stack:** HTML5, CSS3, JavaScript
*   **Features:**
    *   CSS Grid layout for button alignments.
    *   Handles basic operations: addition, subtraction, multiplication, and division.
    *   Real-time display screen updating dynamically based on user clicks.
